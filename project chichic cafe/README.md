# ☕ Chic Chic Cafe (Django Web Project)

## 📌 รายละเอียดโปรเจค

เว็บไซต์ระบบจัดการเมนูร้านกาแฟ พัฒนาโดยใช้ Django Framework
สามารถเพิ่ม แก้ไข ลบ และค้นหาเมนูได้

---

## 🚀 ฟีเจอร์หลัก

* แสดงรายการเมนู
* ค้นหาข้อมูล (ชื่อ / ประเภท)
* เพิ่ม / แก้ไข / ลบ เมนู
* ระบบยืนยันรหัสผ่านก่อนแก้ไขข้อมูล

---

## 🛠️ เทคโนโลยีที่ใช้

* Python
* Django
* Bootstrap 5

---

## ⚙️ วิธีติดตั้ง

1. โหลด โปรเจค

```
https://github.com/sakayasinjud/Project-Chi-Chi-CaFe
```

2. เปิดโฟลเดอร์ใน vs code

```
project chichic cafe
```

3. ติดตั้ง Django

```
pip install django
```

4. รันเซิร์ฟเวอร์

```
python manage.py runserver
```

5. เปิดเว็บ

```
http://127.0.0.1:8000/
```
